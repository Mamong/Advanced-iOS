##6.11　循环控制
命令        用途

break   终止程序，跳出循环

next    跳到下一次循环

redo    在相同的条件下重复刚才的处理

###6.11.1　break
break 会终止全体程序。

###6.11.2　next
使用 next 后，程序会忽略 next 后面的部分，跳到下一个循环开始的部分。

###6.11.3　redo
redo 与 next 非常像，与 next 的不同之处是，redo 会再执行一次相同的循环。
